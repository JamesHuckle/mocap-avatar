## MediaPipe Pose `.task` model

This app uses **MediaPipe Tasks Vision PoseLandmarker** and expects the model file to be present at:

- **`public/models/pose_landmarker_full.task`**

### Download

Download the official **Pose Landmarker Full** `.task` bundle and save it to the path above:

- `https://storage.googleapis.com/mediapipe-models/pose_landmarker/pose_landmarker_full/float16/1/pose_landmarker_full.task`

After adding the file, restart `pnpm dev`.
